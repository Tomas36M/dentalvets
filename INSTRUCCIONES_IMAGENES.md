# Instrucciones para agregar las imágenes

## Imágenes necesarias

Por favor, agrega las siguientes imágenes a la carpeta `/public`:

### 1. Spry Dental (Pasted image 2)
- **Nombre del archivo**: `spry-dental.jpg`
- **Ubicación**: `/public/spry-dental.jpg`
- **Descripción**: Imagen del producto Spry Dental (foto de perrito usando el spry dental - Pasted image 4)
- **Uso**: Página de productos

### 2. Snacks Dentales (Pasted image)
- **Nombre del archivo**: `snacks-dentales.jpg`
- **Ubicación**: `/public/snacks-dentales.jpg`
- **Descripción**: Imagen de los snacks dentales
- **Uso**: Página de productos

### 3. Veterinaria con perrito feliz (Pasted Image 3)
- **Actualización**: Esta imagen ya está siendo usada en el home page desde Cloudinary

## Cómo agregar las imágenes

1. Guarda las imágenes con los nombres exactos mencionados arriba
2. Colócalas en la carpeta `/public` del proyecto
3. Las imágenes se mostrarán automáticamente en la página de productos

## Formatos recomendados
- **Formato**: JPG o WebP
- **Tamaño recomendado**: Máximo 1200px de ancho
- **Peso**: Optimizado para web (menos de 500KB por imagen)
